Spectrum

Spectrum is a 2-8 player twin stick shooter built with Unity. It started out as a 6-person team's final project in 6.073 Creating Video Games of Spring 2016. Since that time, I've worked on it off and on with another teammate, implementing new features and perfecting existing ones across practically every aspect of the game, including sandbox, stat tracking, maps, modes, and developer tools. Screenshots here.

MechGeneral

MechGeneral was a game I worked on in the fall of 2015 as part of CMS.617 Advanced Game Studio. It was built with Unity and features local and online multiplayer support, including matchmaking and private games. I worked primarily on the design and implementation of the gameplay and rules. PingBall

Won Most Entertaining Game for this project in 6.005 Elements of Software Construction. Pinball-esque game that allows for customized boards by parsing a formatted text file. Supports networked boards so that balls can travel to and from boards on other computers.
